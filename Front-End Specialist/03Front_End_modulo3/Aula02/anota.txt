div{
    /* largura pode ser usado w(largura)*/
    width: 200px; 

    /* altura pode ser usado h(altura)*/
    height: 150px; 
    background-color: #ccc;
}

1- por padrão a div ocupara sempre 100% do espaço da  largura da janela
Esse é um padrão para elementos chamados "BLOCK" --> Todo espaço lateral.
ex: Não definir é a mesma coisa que faalar que a largura da div será de 100%

OBS: Medida relativa é uma maneira mais inteligente de trabalhar.

2- Valores definidos em pixels(px), são fixos! Ou seja, eles nunca mudam.
3- Valores definidos em porcentagem(%) tem uma relação com o tamanho do elemento PAI.(onde ele
está inserido).

4- Altura- Caso a altura não seja definida o padrão para definição de altura é auto(automático) - 
Ou seja, conforme você insere conteúdo, ele aumenta.

Seletor de classe --->>> USAR SEMPRE!!!     

O seletor de classes cria uma regra, e essa regra receberá um nome.
Esse nome será usado por qualquer elemento da página!.

**FORMATAÇÃO PADRÃO - E ELA PODE SER REPETIDA**

Para criar um seletor de classe no HTML é:

<p class="nome_para_o_seletor"></p>


Para criar no CSS é:

.nome_para_o_seletor{
    *valores para serem atribuidos a classe*
}

#######################

Seletor de ID - Evitar USAR
você pode usar o atributo id para criar formatações css.

#nomedoid{
    propriedade para ser atribuida
}

na HTML, usar o atributo global id.

<div id="nomedoid"></div>
---------------------------
o id não pode ser repetido na página do HTML, ele deve existir apenas uma vez.
